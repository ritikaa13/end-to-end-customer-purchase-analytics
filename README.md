# End-to-End Customer Purchase Analytics

An end-to-end customer analytics project built using MySQL and Power BI to uncover insights related to customer behavior, retention, segmentation, lifetime value, and churn.

## Tech Stack

- MySQL
- SQL Views
- Power BI
- DAX
